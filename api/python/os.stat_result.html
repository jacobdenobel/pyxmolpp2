os.stat_result class

stat_result: Result from stat, fstat, or lstat.

Contents

  • Reference
    • Methods
    • Special methods
    • Properties
    • Data

This object may be accessed either as a tuple of (mode, ino, dev, nlink, uid, gid, size, atime, mtime, ctime) or via the attributes st_mode, st_ino, st_dev, st_nlink, st_uid, and so on.

Posix/windows: If your platform supports st_blksize, st_blocks, st_rdev, or st_flags, they are available as attributes only.

See os.stat for more information.
